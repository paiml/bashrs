MAKE014: Inefficient shell invocation

Rule: Detect commands that unnecessarily spawn multiple shells

Why this matters: Each shell invocation has overhead. Commands like $(shell cat file) spawn a shell process just to run a simple command. Using Make built-ins or combining commands is more efficient. Multiple separate shell commands should be combined with && or ; when appropriate.

Auto-fix: Suggest more efficient alternatives

§Examples

❌ BAD (inefficient shell usage):

VERSION = $(shell cat VERSION)
FILES = $(shell ls *.c)

✅ GOOD (efficient alternatives):

VERSION = $(file < VERSION)
FILES = $(wildcard *.c)
