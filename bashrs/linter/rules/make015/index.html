MAKE015: Missing .DELETE_ON_ERROR

Rule: Detect Makefiles without .DELETE_ON_ERROR special target

Why this matters: By default, Make leaves partially-built target files when a recipe fails. This can cause corrupted builds. .DELETE_ON_ERROR tells Make to delete the target file if the recipe fails, ensuring clean builds.

Auto-fix: Add .DELETE_ON_ERROR: at top of Makefile
