MAKE020: Missing include guard

Rule: Detect included Makefiles without include guards (double-inclusion prevention)

Why this matters: When Makefiles are included multiple times (directly or transitively), variables can be redefined multiple times, rules can be duplicated, and builds become slower. Include guards (like C header guards) prevent this by ensuring a file is only processed once.

Auto-fix: Add include guard pattern at top of file
