DEVCONTAINER004: Docker Compose without service

dockerComposeFile requires ‘service’ property to specify which service to use.

§Example

❌ BAD (missing service):

{
  "dockerComposeFile": "docker-compose.yml"
}

✅ GOOD (has service):

{
  "dockerComposeFile": "docker-compose.yml",
  "service": "app"
}
