1//! WebAssembly Support for bashrs
2//!
3//! Phase 0: Feasibility Study
4//!
5//! This module implements WASM bindings for bashrs, enabling:
6//! - Browser-based shell script analysis
7//! - Config linting without server round-trips
8//! - Serverless function deployment
9//! - Educational shell script tooling
10//!
11//! ## Architecture
12//!
13//! ```text
14//! Browser/Node.js
15//!     │
16//!     ▼
17//! JavaScript API (wasm_bindgen)
18//!     │
19//!     ▼
20//! Rust WASM Module
21//!     │
22//!     ├─► Config Analyzer (CONFIG-001 to CONFIG-004)
23//!     ├─► Bash Parser
24//!     ├─► Linter
25//!     └─► Virtual Filesystem (via JS callbacks)
26//! ```
27//!
28//! ## Phase 0 Goals
29//!
30//! 1. **Streaming I/O**: Test JavaScript callback performance for streaming output
31//! 2. **Config Analysis**: Port CONFIG-001 to CONFIG-004 to WASM
32//! 3. **Performance**: Benchmark WASM vs native execution
33//! 4. **Memory**: Validate memory usage stays <10MB
34//! 5. **Decision**: Go/No-Go based on streaming I/O feasibility
35
